# 合家云社区物业管理平台
## 安装手册

---

## 1. 环境要求

| 环境 | 版本要求 | 说明 |
|------|---------|------|
| JDK | 21 | 长期支持版本 |
| Node.js | 18+ | 前端构建工具 |
| MySQL | 8.0 | 数据库 |
| Maven | 3.x | 后端构建 |

### 1.1 检查环境

```bash
# 检查JDK版本
java -version

# 检查Node版本
node -v

# 检查MySQL版本
mysql --version

# 检查Maven版本
mvn -v
```

---

## 2. 数据库安装

### 2.1 创建数据库

登录MySQL后执行：

```sql
CREATE DATABASE hejiayun DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2.2 执行建库脚本

```bash
# 方式一：命令行执行
mysql -u root -p hejiayun < backend/src/main/resources/schema.sql

# 方式二：使用数据库客户端（如Navicat、DBeaver）
# 打开 schema.sql 文件，执行全部SQL
```

### 2.3 验证数据库

```sql
USE hejiayun;
SHOW TABLES;
```

应显示9张表：
- hjy_building
- hjy_community
- hjy_owner
- hjy_owner_room
- hjy_room
- hjy_unit
- sys_role
- sys_user
- sys_user_role

### 2.4 导入测试数据（可选）

如果需要测试数据，执行：

```bash
mysql -u root -p hejiayun < backend/src/main/resources/test_data.sql
```

## 3. 后端安装

### 3.1 配置数据库连接

编辑 `backend/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/hejiayun?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password  # 修改为你的MySQL密码
    driver-class-name: com.mysql.cj.jdbc.Driver
```

### 3.2 启动后端

```bash
cd backend

# 方式一：Maven启动（开发环境）
mvn spring-boot:run

# 方式二：运行JAR包（生产环境）
# 先打包
mvn clean package -DskipTests
# 再运行
java -jar target/hejiayun-0.0.1-SNAPSHOT.jar
```

### 3.3 验证后端

访问 http://localhost:8080/api/login ，应返回JSON响应。

## 5. 常见问题

### 5.1 后端启动报错：数据库连接失败

**原因**：MySQL未启动或密码错误

**解决**：
1. 检查MySQL服务是否启动
2. 确认 `application.yml` 中的用户名密码正确

### 5.2 前端页面空白

**原因**：后端未启动，或跨域配置问题

**解决**：
1. 确认后端已启动在8080端口
2. 检查浏览器控制台是否有跨域错误

### 5.3 登录失败

**原因**：数据库中没有初始用户数据

**解决**：执行 `schema.sql` 中的初始数据插入语句

### 5.4 npm install 失败

**原因**：网络问题或Node版本过低

**解决**：
1. 使用淘宝镜像：`npm install --registry=https://registry.npmmirror.com`
2. 确认Node版本 >= 18

## 7. 配置说明

默认端口：**8080**

修改位置：`backend/src/main/resources/application.yml`
```yaml
server:
  port: 8080
```

默认API地址：**/api**（代理到 http://localhost:8080）

修改位置：`frontend/vite.config.js`

默认端口：**5173**

修改位置：`frontend/vite.config.js`
```javascript
server: {
    port: 5173
```

## 8. 生产部署

### 8.1 后端部署

```bash
# 1. 打包
mvn clean package -DskipTests

# 2. 上传JAR包到服务器
scp target/hejiayun-0.0.1-SNAPSHOT.jar user@server:/path/to/app/

# 3. 运行
java -jar hejiayun-0.0.1-SNAPSHOT.jar
```

### 8.2 前端部署

```bash
# 1. 打包
npm run build

# 2. 上传dist目录到服务器
scp -r dist/* user@server:/path/to/nginx/html/

# 3. 配置Nginx反向代理
```

### 8.3 Nginx配置示例

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /var/www/hejiayun;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API反向代理
    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
